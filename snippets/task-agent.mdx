### AI-assisted work, staff experts in the loop

The **LibOps Task Agent** does the technical work. Your team stays in the loop on every change:

*   **Site managers** start from plain text — in the Dashboard or Slack.
*   **The Task Agent** implements the change on a task branch and deploys it to an isolated preview environment.
*   **Engineers** review the diff in a standard GitHub pull request — code, configuration, and dependencies.
*   **Site managers** validate the live preview before the change merges to production.

Every change is transparent, reviewable, and follows the same workflow your team already knows.
